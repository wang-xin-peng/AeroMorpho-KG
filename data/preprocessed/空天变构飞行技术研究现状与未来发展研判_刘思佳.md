# 空天变构飞行技术研究现状与未来发展研判

刘思佳 孟繁敏 解春雷* 聂 璐 闫 溟 刘智惟

**摘 要：** 针对空天变构飞行技术的发展与应用现状进行分析研究认为，当前国内变构飞行的实现途径仍然以机构驱动为核心。提出现阶段变构飞行的高成本特征是其无法顺利向产品化推进的重要限制之一；采取费效比等综合指标方式对变构飞行的代价与收益进行对比衡量；严格控制结构重量或将成为未来推动变构飞行技术工程实用化的关键。

**关键词：** 变构飞行器；变形收益；费效比；机构驱动；结构重量

当传统的固定翼飞行器性能囿于单一气动布局的限制，以变构方式在不同形态之间切换、突破现有飞行边界、可适应不同飞行环境、实现陆海空天多域融合的设计构想益发获得学者与工程师们的青睐，变构飞行器逐渐成为国内外的研究热点。其外形和物理属性根据飞行任务、飞行速度、飞行环境等发生改变，以不同的气动布局、结构形式、形态特征满足不同任务需要，从而实现飞行器在整个飞行阶段的作战效能最优。

变构飞行器具有广阔的应用前景，研究人员对此已基本达成共识。张旭辉、陈树生等认为，面对未来场景中复杂度愈发提升的任务需求，变构飞行器将是重要的发展趋势之一。但变构技术实现难度高，在气动设计、飞行控制、结构机构等多方面均存在诸多难题，例如变构导致飞行器在迎风面出现结构缝隙，高速来流在缝隙中振荡，将造成极端热环境，严重危害飞行安全。并且变构机构的设计与设置势必会导致飞行器系统复杂度提升、结构重量增加，在给飞行器带来部分性能优势的同时，其在总体设计层面的代价也将十分突出明显。

本文旨在讨论推动未来变构飞行方案设计向工程实用化方向发展的技术途径。基于此前变构飞行的研究发展历程与国内当前现有的相关技术积累，分析并提出变构飞行方案设计可采用的收益评价方式，对未来空天变构飞行设计的发展思路做出技术研判。

# 1 研究现状与技术积累

随着传统单一构型的性能天花板逐渐显现，变构飞行成为生成新域新质能力的契机，国内数家高校和科研院所多点发力，已开展近几十年的相关研究。

## 1.1 气动布局设计

气动布局设计包括气动外形的设计、飞行器布局形式的选择、主要气动参数的确定、为满足任务需求而需要采取的气动措施等，是飞行器设计中的重要环节。对于变构飞行器，其气动布局的性能设计与评估又可粗略分为两大重要模块，包括基准气动外形的选择与变构过程的设计。基准外形是总体设计中的首要难点，决定了变构飞行器在常规飞行模式下的飞行性能，应根据变构飞行器的多任务剖面，选取气动外形的“最大公约数”，以此为基础开展变构形态与变构策略等动态过程设计。变构的动态过程设计则是变构飞行器与常规固定外形飞行器的显著区别所在，变构过程中的非定常气动效应与焦点位置的大幅移动为在极大程度上为总体设计增加难题。

张旭辉等将变构飞行器设计难的原因归结为“多维度、多尺度、强耦合”三方面：在设计维度上，变构飞行的变化方式涵盖了在空间与时间层面的多维度连续变化，非定常效应明显，设计维度显著增高，需要考虑的设计工况也会存在相应量级上的提升；在设计尺度上，其连续变化同时存在于飞行器宏观外形的大尺度改变与缝隙引起边界层层面的小尺度变化，多尺度变化使非定常现象更加复杂；此外，考虑到变构飞行在时间、空间多维坐标下的形态变化，流热固耦合等复杂多物理场效应相继出现。综合来讲，变构飞行在气动布局方面的设计难题可归结为两个原因：非定常效应突出与焦点变化导致的操稳特性弊端。

陈树生等基于气动布局与总体需求的协调关系对变构飞行器气动布局设计的关键技术难点进行

梳理与分析，认为在该专业领域，时变空气动力学效应评估、气动优化设计与多学科耦合一体化设计等技术将是未来所需重点聚焦的研究方向。其中，动态气动效应评估是变构飞行器外形设计的基础，当前亟需建立时变非线性气动力模型以及高精度实时快速评估方法，为缩短变构飞行器气动布局设计周期以及建立后续结构设计、控制系统设计和变构决策设计等提供气动数据基础。

吕侦军等对变构飞行的关键气动技术进行分析指出，气动布局的选择与设计需从总体需求、稳定控制、结构实现等多专业角度考虑设计约束，总结了现阶段实现变构飞行器动态气动性能预测的风洞试验与数值仿真相关技术，并梳理了变后掠、折叠翼、伸缩翼、柔性翼4种变构方式对飞行器气动性能的影响，并将4种变构方式的优缺点进行了总结（表1）。

## 表1 四种变构方式对比

| 变构方式 | 优点 | 缺点 |
| --- | --- | --- |
| 变后掠 | 随速度变化后掠角，适用于兼顾低速和高速性能的宽域飞行 | 变形驱动机构和局部结构加强，质量代价大 |
| 折叠翼 | 适用于强尺度约束条件，折叠机构易实现，质量代价小 | 在高速高动压下翼面载荷大，折叠/展开困难 |
| 伸缩翼 | 多用于低速飞行器，变形方式简单易实现 | 为嵌套结构，占用翼面空间，容积利用率低 |
| 柔性翼 | 可以根据飞行环境自适应改变外形，可用于未来智能变形飞行器 | 变形幅度小，目前的柔性变形材料无法适用于高速飞行器的高热环境 |

针对变构过程中的非定常气动效应，国内白鹏的团队聚焦该问题开展了一系列研究。区别于已有的准定常气动性能研究工作，徐国武等对非定常连续变形过程中的气动性能变化初步开展研究，针对无粘不可压流场中儒科夫斯基翼型厚度、弯度的连续变化过程，分析不同变形幅度、变形速度以及变形加速度对非定常升力系数的影响情况。陈钱等针对改变后掠角的变构方式，对风洞试验过程中表现出的非定常气动特性从形成机理层面进行了深入分析，并将变后掠过程中的非定常性用3种物理效应进行解释，分别为流场迟滞效应、附加运动效应和固壁牵连效应，从机理研究的角度为后续开展变构飞行器变后掠过程设计的气动特性建模奠定基础。刘欣煜等以一种滑动蒙皮变后掠翼飞行器为研究对象，基于非定常试验数据，开展了变后掠过程中的非定常气动力建模工作，所建立的多项式模型较好地描述了不同变形周期下作用在飞行器上的非定常气动力。

## 1.2 结构机构与蒙皮材料

结构与材料是实现变构飞行的物质基础，变构飞行的研发设计、生产制造与结构、机构、蒙皮材料有着不可分割的联系。不同的变构飞行方案设计对结构与材料提出了不同程度的要求。概括地说，变构的实现方式有两种：一种仍然以机构驱动为主，但随着机械工业水平的提升，相比过去的变构方式也必然呈现出高度自动化的特征；另一种则以智能材料为核心，这种变构方式更符合设计者们希望飞行器表面始终光滑、连续、无缝过渡的设计需求。

从发展趋势来看，以智能材料为核心的柔性变构也是国内外研究者们共同希冀的发展方向。在变构特征方面，国外的变构飞行设计方案经历了从刚性变体、设计主动气动弹性机翼、到聚焦柔性变形的不同发展阶段。刚性变构主要指20世纪50年代—80年代间，此时出现了包括X-5、XB-70、F-111、Su-17、MIG-23、Su-24、Tu-22M、F-14、B-1等经典机型，借助于机械结构驱动，设计方案包括变后掠翼、变展长等，变构方式以刚性变构为主。而在这之后，发展主动气动弹性机翼实际是从追求机翼刚度、避免弹性变形到反过来利用机翼结构的柔性、主动产生弹性变形、从而有效控制飞机运动的认识与设计转变过程，初步引入智能材料与结构，将智能材料与控制驱动系统相结合，使变构飞行逐渐具备自适应甚至智能化的能力。到了柔性变构阶段，对智能材料与结构的应用进一步加强，以形状记忆合金（Smart Memory Alloy, SMA）替代传统铰链，实现对机翼扭转变形的驱动。在此期间，NASA联合DARPA等机构先后大力推动多个研究验证项目，包括“变形飞行器计划（The aircraft morphing program）”、“任务自适应机翼（Mission Adaptive Wing, MAW）”、“主动柔性机翼（Active Flexible Wing, AFW）”、“主动弹性机翼（Active Aeroelastic Wing, AAW）”、“智能机翼（Smart Wing）”、“变形飞行器结构（Morphing Aircraft Structures, MAS）”等，并且在研究工作开展期间，多个飞行验证演示试验取得成功，但在这些项目之后，相关的研究工作却逐渐偃旗息鼓。纵观国外变构飞行的发展历程，对以智能材料为基础的

柔性变构的追求愈发提升，但从技术实现的角度来看，刚性变构阶段实现了大批型号生产，因此依然是以机械结构驱动的变构方式技术成熟度最高，而柔性变构阶段的多个项目最后也逐渐不了了之。

相对来说，国内在变构飞行技术的研究领域起步即较晚，但发展阶段的演进过程大体与国外有相似之处。在以刚性变构为主的20世纪60—70年代，也曾对变后掠翼技术聚力攻关，并将其用于“强-6”，此后从20世纪90年代起，在国自然等相关科研项目的引导下，近10～20年内也在智能材料与结构领域逐渐取得了长足进展。梁尚军等聚焦变体飞机结构领域，围绕国内外较受关注的热点专题，开展了相关综述，对国内高校的研究进展初步进行总结。

肖洪等按照翼面内变形与翼面外变形的方式对飞行器变形翼进行分类，其中翼面内变形包括变后掠、变展长、变弦长，翼面外变形包括变扭转、展向变弯曲、弦向变弯曲和翼型变厚度。对每种分类，又进一步按照机构变形和结构变形的形式详细阐述，其中机构变形指通常由构件和运动副相对运动来实现变构的驱动方式，而结构变形则指通常由弹性元件、智能材料驱动实现变构的设计方案。基于对飞行器变形翼研究现状与发展趋势的分析研判指出，为推动变形翼技术发展与应用，未来着手工作研究应重点关注如下5个方面：高刚度多维度变形机构设计理论与方法、大承载连续光滑变形蒙皮结构设计、高效能驱动器及分布式驱动技术、变形翼机构-蒙皮-驱动一体化设计与优化以及融合环境的变形翼设计与验证方法。

张家应等按照大尺度、中等尺度和小尺度的三类划分方式对变体飞行器结构技术进行分类，三种尺度中大尺度变构涉及变展长、变后掠角、变机翼面积和形状等，中等尺度变构包括变翼型弯度、变翼型厚度、变机翼扭转等，小尺度类型则涵盖了鼓包设计等局部变形方式。在给出飞行器主要变构方式的基础上，分析了变构设计在变形蒙皮、驱动技术、变形机构等技术方面的研究重点与开展方法，指出驱动系统是实现变构的关键组成，当前迫切需要强大、轻量以及紧凑的解决方案。为保证变构所能实现的气动增益远大于附加质量所引起的弊端，确保变构飞行器整体质量的减轻尤为重要。

智能材料驱动器和柔性蒙皮作为变构飞行器设计的关键技术，杨森等对其研究进展进行了总结分析。详细介绍了形状记忆合金、压电材料、磁致伸缩材料三种典型智能材料驱动器的应用现状，对相关性能进行对比，同时分析了基于材料弹性和基于结构的柔性蒙皮发展现状指出，目前只有少数研究成果在工程中得到了实际应用，更多数还处在实验室研究阶段，并且仍存在部分待解决的问题。例如，SMA变形依赖温度变化，但其加热与冷却过程十分缓慢，响应率特性低，无法满足实时变构需求。而大部分的柔性蒙皮目前仍无法用于高速领域，在满足弹性与大变形的同时，仅能承受低速飞行时的气动载荷，并且对于材料老化与疲劳性能的分析研究尚未充分开展，与工程应用仍存在差距，提出当前智能材料驱动器的研究多使用单一材料，驱动器性能囿于智能材料本身特性限制，从工程应用的角度出发，采用机械机构与智能材料驱动器相结合的方法或将更具可行性。

综合来看，当前国内对智能材料与结构的研究工作成果颇丰，但从技术成熟度的角度来讲，到实现工程落地仍有一定距离。相对来说，机械结构驱动的完成度最高，但其缺点同样十分明显，机构笨重，占据了有效载荷的质量与空间，且维护成本高，使变构飞行的使用成本大幅升高，严重削弱变构方案设计的预期收益。短期内，想推动变构飞行技术向工程实用化方向进一步转化，仍需从结构机构设计方面入手，对变构机构导致的结构重量增加实现严格控制，以尽可能保留甚至争取变构飞行的收益。

## 2 优势、代价与收益评价方式

在以往的公开文献中，常以突出变构带来的收益为主，而对于变构给飞行器系统所带来的负面影响则评价相对较为模糊。采用合理的评估指标，对变构飞行的必要性与可行性进行定性甚至定量分析，目前国内在该方向的研究仍略显空白。对于变构所满足的高性能需求与变构所导致的高使用成本对立所形成的矛盾，在两者之间权衡选择、折衷妥协，如何同时兼顾对性能与效率提升的追求和对生产、使用与维护成本的控制，如何从必要性与可行性两方向对变构飞行的总体设计方案进行论证，找到合适的发展赛道与相宜的发力点，是工业部门应当率先思考的重要问题。

变构飞行的优势突出体现在可通过改变外形来改善飞行性能，进而满足多任务飞行需求，但与此同时，变构所带来的代价也十分明显。以表1为例，

在实际工程设计中，对效益的考量是必不可少的参考因素。在美国空军与NASA合作开展的任务自适应机翼（Mission Adaptive Wing, MAW）演示验证项目中，AFTI/F-111改装机（图1）成功试飞，美方却依然决定不将该项技术投入使用，与结构重量在使用和维护方面导致的高成本消耗不无关系。F-14（图2）曾名重一时，但最终遭遇提前退役，变后掠翼技术的低可靠性与高维护成本也是缘故之一。在低经济性消耗愈发受到重视的现代社会，变构技术当前昂贵的使用成本似乎与发展趋势相悖。效能提升有限，实用性偏低，技术成熟度不足，可以说高昂的成本代价是限制变构技术进一步落地的重要原因。

从现阶段的变构飞行方案来看，气动布局与相关策略设计或将决定变构飞行的性能与收益提升上限，然而结构机构的设计与实现方式则是控制代价与成本的核心要素。从气动的角度论需求，从结构的角度论成本。单纯追求气动效率与飞行性能的极致提升显然会使飞行器系统在重量、可靠性等多方面付出的成本代价急剧增加。此时，应当采用诸如效费比、性价比等相似性质的综合评价指标，筛选出效果好、易实现、成本低的变构设计方案。同时，以费效比的收益评价方式指导变构飞行设计方案的筛选与优化，对工业部门加以指导，可将气动专业、结构专业、材料专业相结合。从气动的角度出发，探讨变构设计带来的升阻比增量、升力系数提高、阻力系数下降；从结构专业的角度出发，回答实现该变构方案设计所需的时间和经费成本。以综合评价指标对变构的收益做出量化，对设计部门与决策机构均可提供较高的参考意义。

# 3 未来发展研判

迄今为止，国内聚焦变构飞行技术多点发力，基础研究已有相应程度的积累，但离应用于工程实际设计尚有一定距离。未来如何牵引变构飞行技术的研究成果向产品方向落地转化，是工业部门应当肩负的责任，也是工业部门的优势所在。

目前而言，国内的变构方案设计仍然以机构驱动为核心，蒙皮材料的作用仍然以维持光滑外形为主。现有变构飞行设计方案的常见弊端之一即为驱动机构占据了大量可使用空间，同时又增加了飞行器的结构重量，直接导致飞行器的重量效率与空间利用率严重降低。可以说，如果变构飞行器不能进一步在空间与重量方面控制成本甚至争取收益，其相比于传统的常规固定布局飞行器的性能优势则将十分难以凸显。因此，现阶段想要继续推动变构飞行向工程实用化方向发展，严格限制飞行器的结构重量是必备条件。

# 4 结束语

从气动布局设计、结构机构与蒙皮材料多专业角度对国内变构飞行相关的技术积累进行分析总结，肯定基础研究的重要性与良好发展生态，但也指出当前的技术成熟度仍不具备推动变构飞行器向实际工程应用转化的程度。目前的变构方案设计仍然以机构驱动为核心，结构重量增加、系统复杂度提升等不利因素导致的飞行器成本大幅增长也是限制变构飞行器产品化的原因之一。在变构飞行设计的必要性方面，提出采用费效比等综合评价方式对方案的代价与收益进行对比衡量，并指出，严格控制飞行器的结构重量是推动变构飞行进一步发展的必经阶段。
